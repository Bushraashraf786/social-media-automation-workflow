# Social Media Automation Workflow

Automate your social media post notifications with Make.com!  
This workflow monitors Google Sheets for new posts and sends a formatted Gmail alert automatically.  
It helps teams stay updated, saves time, and prevents missed posts.  
A beginner-friendly, scalable, and time-saving automation solution.

This project uses **Google Sheets**, **Make.com**, and **Gmail** to create a simple social media notification system.

Whenever a new entry is added to the Google Sheet (Date, Platform, Content), the workflow on Make.com automatically sends an HTML-formatted email alert.  
This is ideal for content creators, educators, small businesses, and anyone who wants to save time and avoid manual reminders.
